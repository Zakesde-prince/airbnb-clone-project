# Airbnb Clone Project

## Project Overview

This is a clone of the Airbnb platform, built as a full-stack web application. The goal is to allow users to search, view, and book properties online.

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Version Control**: Git, GitHub
- **Design**: Figma for UI/UX mockups
- **Backend (later)**: APIs and database logic
